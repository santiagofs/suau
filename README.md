# Suau

A simple React 19 + Vite + TypeScript project with a centered SVG on a black background.

## Getting Started

Install dependencies:

```bash
pnpm install
```

Run the development server:

```bash
pnpm dev
```

Build for production:

```bash
pnpm build
```

## Adding Your SVG

Replace the placeholder SVG in `src/logo.svg` with your SVG content. The SVG will automatically expand to fill the available space while maintaining its aspect ratio.
